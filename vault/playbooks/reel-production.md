# Playbook: client reel production

Input: client name + topic/occasion. Load `clients/<name>.md` FIRST (voice + formats).

1. **Hooks**: write 3 hook options in the client's voice (≤8 words each). Pick strongest.
2. **Script**: 15-30s structure — hook (0-2s) → payoff tease → 2-3 beats → CTA. ≤60 words.
3. **Visuals**: Higgsfield MCP — `generate_video` for generative shots, or
   `shorts_studio_create` for full short-form assembly. Product/venue photos beat pure AI
   for food/venue clients; use AI for b-roll and transitions.
4. **Caption**: client voice, first line = hook restated, 3-5 niche hashtags, no hashtag walls.
5. **QC loop**: watch/view the output. Wrong vibe → fix the prompt, not the caption. One retry max
   before showing options to Alex.
6. Log: append deliverable to the client file's Open items (mark shipped).

Cost rule: batch generations; preview cheap/low-res first where the tool allows.
