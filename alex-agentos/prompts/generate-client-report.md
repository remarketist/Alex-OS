# Prompt: Generate a client report

---
Generate this period's report for <CLIENT NAME>. Use the `monthly-client-report` skill. Read `reporting.md` and `CLAUDE.md` (for KPIs and tone).

Inputs I'll provide: this period's metrics (spend, leads, CPL, bookings, cost-per-booked, revenue, organic), last period's numbers, and the work we delivered.

Deliver (via `templates/monthly-report-template.md`): a 30-second plain-language summary, the numbers table with deltas vs last period and target, what changed, what it MEANS (interpretation — never skip), what we did, 1-3 recommendations each with expected impact, and next period's priorities. Lead with cost-per-booked/revenue, be honest about misses, one decision per recommendation.
