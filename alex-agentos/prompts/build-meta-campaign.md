# Prompt: Build a Meta campaign

---
Build a Meta Ads plan for <CLIENT NAME>. Use `meta-ads-strategist` and the `meta-ad-angle-generator` skill.

Context: offer = <OFFER + PRICE>; buying cycle = <impulse/considered>; destination = <landing/WhatsApp>; monthly budget ≈ <€X>. Pull the differentiation angle from `competitors.md` if present.

Hand hero creative to `image-video-prompt-engineer`.
