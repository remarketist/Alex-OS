# Restaurant Content System

Hospitality content wins on appetite, atmosphere, and personality — filmed fast with a phone by staff. Build a system, not one-off posts: one shoot day → 1-2 weeks live.

## Inputs
- Venue type, vibe, signature items, target guest, city. Read `brand-voice.md` / `content-examples.md` if present.

## Content pillars (pick 4)
- **Food hero** — the dish as the star: sizzle, pour, cheese-pull, steam, plating. Appetite in the first second.
- **Vibe / atmosphere** — the room, lighting, music, crowd energy; sells the night out, not the dish.
- **People / behind-the-scenes** — chef, bartender, staff personality; the human hook.
- **Entertainment / trend** — funny food moments, absurd situations (great for ramen/cocktail brands), platform trends adapted to the venue.
- **Social proof / occasion** — packed nights, reactions, events, "who to bring".

## Produce
1. **Reel concepts (10-12)** mapped to pillars — each with a 1-second hook (visual + text) and why it stops the scroll. All filmable with a phone by staff.
2. **Hooks (12-15)** — appetite-, curiosity-, or personality-led.
3. **Captions (6-8)** in the venue's exact voice (witty/absurd/elegant per client) — first line = second hook, end = one action (book / visit / tag).
4. **Weekly calendar** — which pillar posts which day, cadence realistic for a busy venue (3-5×/week), with the format (reel/story/carousel).
5. **Shoot list** — the ~10 clips to capture in ONE shoot day that fill the calendar (batch thinking).
6. **Repurposing** — each clip → reel + story + potential ad (hand hero clips to `meta-ad-angle-generator`).

## Output
The 6 sections above as a plug-and-play weekly system the venue can run with a phone.

## Never
- Generic "foodie" content that fits any restaurant. Match THIS venue's personality.
- Ideas needing a crew, studio, or a full day per clip.
- Beauty shots with no hook — appetite/personality must hit in the first second.
