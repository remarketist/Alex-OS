# Test: Events venue Meta Ads

## Scenario
An events venue with a garden wants wedding/event leads via Meta. Considered purchase, lead form → sales follow-up.

## Run
Use `prompts/build-meta-campaign.md` → `meta-ads-strategist` + `meta-ad-angle-generator`. Offer: venue tours / date-hold; destination: lead form + WhatsApp; considered cycle.

## Pass bar
- Objective tied to booked venue tours / qualified enquiries — not "leads" or "reach".
- Campaign structure supports clean learning (consolidated, not fragmented).
- Audience reasoning stated (broad/Advantage+ for a considered local purchase).
- 3-6 DISTINCT angles (aspiration, proof/real weddings, urgency of date availability, garden USP) — not one angle reworded.
- Hook bank stops the thumb; 2-3 primary texts written; testing matrix with a kill/scale rule.
- Retargeting flow for enquired-not-booked (proof, date urgency, tour invite).
- Tracking: Pixel+CAPI, the right event, cost-per-qualified-enquiry.
- Explicitly ties ads → offer → landing/WhatsApp.

## Fail if
- Broad targeting with no reasoning; angle variations instead of distinct angles; ads detached from offer/page; ignores creative fatigue.
